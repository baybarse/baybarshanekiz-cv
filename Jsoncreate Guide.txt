# jsonresume için Kurulum 

vs code açılır ve istenilen dizine girilir. Terminal açılır ve git bash açılır.
Şuradaki Installation adımları izlenebilir: https://docs.jsonresume.org/cli


Not: Dosyaların yollarında boşluklu dosyalar olmamalıdır.


Yada adım adım şunlar yapılır:

Ön gereksinimler:
- Node kurulu olacak
- Git kurulu olacak

Json dosyası dizine eklenir veya düzenlenir.

node --version
# v18.0.0 or higher recommended

npm install -g resume-cli
resume init
resume validate
# stackoverflow için:

npm install jsonresume-theme-stackoverflow 
resume serve --theme stackoverflow

# veya stackoverflow'dan farklı diğer template'ler: https://github.com/jsonresume/jsonresume.org?tab=readme-ov-file#themes



jsonresume-theme'leri listelemek için:
npm search jsonresume-theme

----

Meta Bölümünde Sections Sıralaması Örneği, Burada Hangi Sections Verilirse Site O Şekilde Olur:

 "meta": {
    "theme": {
      "sectionOrder": [
        "basics",
        "work",
        "skills",
        "education",
        "projects"
      ]
    }
  }
 
---
Bölümler(Sections): 
basics, work, education, skills, projects, volunteer, awards, certificates,  publications, languages, interests, references
